# openpharma Server 0.2.1

## Description
This package is the backend application for openpharma


## Build Setup

``` bash
# install dependencies
npm install

# Compiles and hot-reloads for development @ localhost:3000
npm run dev

# Compiles and minifies for production
npm run build
```
